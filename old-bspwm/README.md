## Bahan-bahan

- WM: bspwm, xfwm4
- Screenshot: scrot, flameshot
- Keybindings: sxhkd
- Terminal: alacritty
- Image viewer: viewnior
- File Manager: thunar, ranger
- Status Bar: polybar, tint2
- Text Editor: neovim
- Set Wallpaper: nitrogen
- Music Player: ncmpcpp
- Icons: papirus custom
